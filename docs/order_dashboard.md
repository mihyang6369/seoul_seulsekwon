# 슬세권 지수 시각화 대시보드 구축 작업지시서

## 1. 프로젝트 개요
- **목적**: 서울시 내 주요 편의시설 데이터를 바탕으로 특정 위치의 '슬세권(슬리퍼 생활권) 지수'를 산출하고, 이를 시각화하여 사용자에게 직관적인 정보를 제공함.
- **핵심 가치**: 주거지 인근의 편의성(교통, 상업, 문화, 의료, 녹지 등)을 정량화하여 비교 가능한 지표 제공.

## 2. 데이터 소스 (Data Source)
`data/cleaned` 폴더 내의 정제된 CSV 파일들을 사용한다. 각 파일은 `위도`, `경도` 정보를 포함하고 있어야 한다.

- **생활/편의**: `starbucks_seoul_cleaned.csv` (스타벅스), `sosang_seoul_cleaned.csv` `large_scale_shop_seoul_cleaned.csv` (편의점, 세탁소, 마트 등), 'large_scale_shop_seoul_cleaned.csv' (대형마트, 백화점 등)
- **교통**: `bus_station_seoul_cleaned.csv`, `metro_station_seoul_cleaned.csv`
- **의료**: `hospital_seoul_cleaned.csv`, `sosang_seoul_cleaned.csv` (병원, 약국)
- **안전/치안**: `police_seoul_cleaned.csv`
- **교육/문화**: `library_seoul_cleaned.csv`, `bookstore_seoul_cleaned.csv`, `school_seoul_cleaned.csv`
- **자연/여가**: `park_raw_cleaned_revised.csv`
- **금융**: `finance_seoul_cleaned.csv`
점수를 부여하고

### [카테고리별 가점 기준]
- **카테고리별 기준 가점**
생활/편의 점수 = (min(실제 시설 개수,max) / max))*카테고리 가중치(30)
교통 점수 = (min(실제 시설 개수,max) / max))*카테고리 가중치(20)
의료 점수 = (min(실제 시설 개수,max) / max))*카테고리 가중치(15)
안전/치안 점수 = (min(실제 시설 개수,max) / max))*카테고리 가중치(10)
교육/문화 점수 = (min(실제 시설 개수,max) / max))*카테고리 가중치(5)
자연/여가 점수 = (min(실제 시설 개수,max) / max))*카테고리 가중치(15)
금융 점수 = (min(실제 시설 개수,max) / max))*카테고리 가중치(5)

- 사용자가 슬라이더로 가점을 조절했을 경우 해당 값을 따른다.

## 3. 사용 라이브러리

### 📍 위치 설정 방식
- 카카오 api를 활용하여 검색한 장소의 위도, 경도를 기반으로 지도 표시 및 지도 클릭 시 해당 위치로 즉시 재분석(`st.rerun`)하는 기능 포함.
- api는 최초 1회만 호출하고, 이후에는 세션에 저장된 값을 사용하도록 함.
- 지도에 마커를 표시하고, 마커를 클릭하면 해당 위치로 지도를 이동시킴.
- api key는 .env 파일에서 관리하고, .env 파일이 없을 경우 안내 메시지 출력.

### ⚙️ 가중치 및 지수 산출
- 100점 만점을 기준으로 한 정량적 점수 산출 및 S~C 등급 부여. 작업지시서의 공식을 엄격히 준수.

### 🗺️ 지도 및 시각화 인프라
-`st_folium`을 사용하여 인터랙티브 지도 제공. 레이더 차트 중심의 시각화. `st.spinner`를 활용한 UX 처리.

## 3. UI/UX 디자인 특징
- netlify 배포를 고려하여 디자인.
- 메인 화면에는 주소 검색창만 표시하고, 검색 버튼을 클릭하면 분석 결과가 표시되도록 함.
- 분석 결과는 메인 화면에 표시하고, 지도에는 마커를 표시함.
- 마커는 각 카테고리별로 이모지로 표시할 것.
