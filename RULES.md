## 1. 독립된 경로 사용
- 각 프로젝트별로 독립된 경로를 사용합니다.
- 외부 폴더나 절대 경로에 의존하지 않고, 해당 프로젝트 폴더를 기준으로 하는 상대 경로를 사용합니다.
- 데이터 파일은 프로젝트 폴더 내에 `data/raw` 폴더에서 불러오고 결과 파일은 데이터 파일은 `data/cleaned` 폴더에 저장합니다.
- 파일 생성에 사용된 .py 파일은 `scripts` 폴더에 저장합니다.

## 2. 가상환경 통합 사용
- 각 폴더마다 개별 가상환경을 만들지 않고, 최상위 루트 폴더의 `.venv` 가상환경을 공통으로 사용합니다.
- 패키지 관리는 루트 폴더의 의존성 설정을 따르거나 필요 시 통합하여 관리합니다.
